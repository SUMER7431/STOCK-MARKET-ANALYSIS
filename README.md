Stock Market Analysis-----------

Project Overview----
Real time stock market analysis of major tech companies using Python and Yahoo Finance API.

Tools Used----
  -Python
  -Pandas
  -Matplotlib
  -yfinance API
  -Jupyter Notebook
Stocks Analyzed----
  -Apple (AAPL)
  -Google (GOOGL)
  -Microsoft (MSFT)
  -Amazon (AMZN)
Key Insights----
  -Apple stock grew over 300% from 2020 to 2024
  -All 4 tech stocks show strong upward trend
  -50 and 200 day moving averages confirm bullish trend
  -Daily returns mostly stay between -5% and +5%
  -2023 was best performing year for Apple
Visualizations----
  -Apple Stock Price Chart 2020-2024
  -Moving Averages Chart (50 and 200 day)
  -Tech Stocks Comparison Chart
  -Daily Returns Distribution
  -Yearly Average Price Chart
Data Source----
  -Yahoo Finance API via yfinance Python library
